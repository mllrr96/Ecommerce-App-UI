## 🚀 Installation & Setup

1. **Clone the repository**
   ```sh
   git clone https://github.com/mllrr96/ecommerce-app.git
   cd ecommerce-flutter
   ```

2. **Install dependencies**
   ```sh
   flutter pub get
   ```

3. **Run the app**
   ```sh
   flutter run
   ```

Make sure you have **Flutter installed** and a connected emulator or device.
